# Exam Flow

> From exam creation to student attempt and results.

---

## Flow Diagram

```text
CREATION                     ASSIGNMENT                   ATTEMPT                    RESULTS
────────                     ──────────                   ───────                    ───────

SuperAdmin:                  Assigns to        
PYP/Grand Tests    ──────►   Institutes        ──────►   
                                               
Institute:                   Assigns to                  Student                    Auto-grade
Pattern Exams      ──────►   Batches           ──────►   Takes Test     ──────►    or Teacher
                                                                                    Review
Teacher:                     Assigns to
Quizzes/Tests      ──────►   Batches           ──────►   
```

## Step-by-Step

1. **Creation**: SA/Institute/Teacher creates exam with questions
2. **Assignment**: Exam assigned to target batches with schedule
3. **Visibility**: Students see exam in Tests list when scheduled
4. **Attempt**: Student opens Test Player, answers questions
5. **Submit**: Responses saved, timer ends or manual submit
6. **Grading**: Auto-grade for objective, manual for subjective
7. **Results**: Score available based on exam settings
8. **Analytics**: Data feeds into My Path recommendations
