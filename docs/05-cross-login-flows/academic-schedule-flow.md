# Academic Schedule Flow

> Planning, teaching confirmation, and progress tracking cycle.

---

## Flow Diagram

```text
INSTITUTE                    TEACHER                      INSTITUTE
─────────                    ───────                      ─────────

1. Hour Setup:               
   Allocate hours            
   per chapter               

2. Generate Plan:            
   Auto-calculate     ────►  3. View Expected         
   chapter-weeks             Progress

3. Publish Plan:             
   Lock schedule      ────►  4. Teach Classes
                             
                             5. Confirm Teaching   ────►  6. Track Progress
                             (topics taught)              (completion %)
                             
                             6. Bulk Confirm       ────►  7. Drift Analysis
                             (pending classes)            (variance tracking)
```

## Confirmation Flow

1. **Teacher teaches** scheduled class
2. **Confirms topics** covered (or marks "Not Taught")
3. **Institute sees** updated progress percentage
4. **Drift calculated** as variance from plan
