# Homework Flow

> Assignment creation to student submission and grading.

---

## Flow Diagram

```text
TEACHER                      STUDENT                      TEACHER
───────                      ───────                      ───────

1. Create Homework:          
   ├── Type (Practice/       
   │   Test/Project)         
   ├── Instructions          
   └── Due Date              

2. Assign to Batches  ────►  3. Sees in Dashboard         
                             (pending homework)

                             4. Opens Submission
                             ├── Practice: File/Text
                             ├── Project: Multi-file
                             └── Test: Opens Player

                             5. Submits          ────────►  6. Reviews Submission
                                                            ├── Practice/Project:
                                                            │   Manual grade
                                                            └── Test: Auto-graded

                             7. Sees Grade/Feedback  ◄────  8. Provides Feedback
```

## Type-Specific Flows

| Type | Student Action | Teacher Review |
|------|----------------|----------------|
| Practice | Upload file/text | Manual grade + feedback |
| Project | Multi-file upload | Manual grade per section |
| Test | Take quiz in player | Auto-graded, view results |
